# i18n-quarkusio

[https://i18n-quarkusio.github.io/](https://i18n-quarkusio.github.io/)

Translation project for [https://quarkus.io/](https://quarkus.io/)

## Build

### Prerequisites

* po4a
* ruby
* bundler

### Build translated documents

```bash
bin/build
```
