# i18n-quarkusio

Translation project for [https://quarkus.io/](https://quarkus.io/)

Japanese: [https://ja-i18n-quarkusio.surge.sh/](https://ja-i18n-quarkusio.surge.sh)

## Build

### Prerequisites

* po4a
* ruby
* bundler

### Build translated documents

```bash
bin/build
```
